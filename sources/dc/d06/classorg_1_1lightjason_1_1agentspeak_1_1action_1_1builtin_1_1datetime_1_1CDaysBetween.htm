Detailed Description

The actions returns the number of days between two date-time objects, the action never fails

[D1|D2] = datetime/daysbetween( DateTime1, DateTime2, DateTime3, DateTime4 );
Note
a positive value will be returned iif the first date-time item is before the second one, a negative value will be returned iif the first date-time item is after the second date-time item
